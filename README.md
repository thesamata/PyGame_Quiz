# 🛡️ PyGame Quiz: Savaşçının Yolu

Python, Tkinter ve Pygame kullanılarak modern bir arayüzle geliştirilmiş hikaye tabanlı bilgi yarışması.

## 🚀 Hızlı Başlangıç

1. **Bağımlılıkları Yükle:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Oyunu Başlat:**
   ```bash
   python Game.py
   ```

## 🎮 Öne Çıkanlar
- **Modern Görsel Tasarım:** Yenilenmiş fontlar, renkler ve ikonlar.
- **Hikaye ve Animasyon:** Patronla (Boss) diyaloglar ve akıcı GIF animasyonları.
- **Dinamik Sesler:** Doğru ve yanlış cevaplar için ayrılmış ses efektleri.
- **Zorluk Seviyesi:** 3 can hakkı ve 10 soruluk havuzdan rastgele gelen sorular.
